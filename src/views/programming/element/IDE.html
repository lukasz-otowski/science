Aplikacje

kompilatory

Kiedy kod źródłowy jest skończony, musi być wykonany proces kompilacji, czyli tłumaczenia/translacji kodu napisanego w języku programowania, na kod maszynowy zrozumiały dla komputera i zapisania go w osobnym pliku.

Intepretatory

?---Interpretatory, to programy w których kod jest wykonywany zaraz po translacji, bez wykonywania procesu kompilacji.---?

Kompilatory są podzielone na dwie grupy, kompilatory języków wysokiego poziomu, oraz asemblery, kompilatory języków symbolicznych/niskiego poziomu.

Konsolidator

Jest to program który podczas kompilacji wykonuje proces linkowania/konsolidacji, czyli dołączania bibliotek, i plikó obiektowych, potrzebnych aplikacji do prawidłowego działania.
